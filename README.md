# Tarea4HD - Diego Urbano - Módulo: Comparador de 4 bits (MAYOR, IGUAL, MENOR)
Este módulo de lógica combinacional compara las magnitudes de dos números binarios de 4 bits de forma paralela. Su función es evaluar ambas entradas y activar una única salida lógica para indicar de forma automática si el primer número es estrictamente mayor, igual o menor que el segundo.
